# Kraken

Kraken is a universal price tracker that lets you monitor products on almost any website.

Track price drops, monitor stock availability, and receive alerts when products reach your target price.

## Features

* Track products from any website
* Price drop notifications
* Back-in-stock alerts
* Price history and trends
* Product dashboard
* Automatic selector recovery

## How It Works

1. Install the Kraken extension
2. Select a product price on any website
3. Configure your alerts
4. Receive notifications when conditions are met

## Tech Stack

**Frontend**

* NextJS
* TypeScript
* Tailwind CSS
* Shadcn UI

**Backend**

* Node.js
* Express
* Prisma
* PostgreSQL

**Automation**

* Playwright

## Getting Started

```bash
npm install
```
